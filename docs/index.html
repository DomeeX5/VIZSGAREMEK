Webshop

This website is based on NestJS backend, Prisma ORM for database handling, and React frontend.

Installation

Node

Run npm install in the terminal, to install every package that is neccessary.
To start, run these commands in the terminal:

cd apps/api
nest start

Database

For the database, you have to install XAMPP, and configure it.
The database should run on port 3306.
